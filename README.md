## 소개

fastImageViewer는 다음 문제를 해결하기 위해 만든 도구입니다.

- 수백 장의 RAW/JPEG가 들어 있는 ZIP 아카이브를 외부 압축 해제 없이 바로 열어 빠르게 넘겨봐야 할 때
- Canon CR3(RAW) 파일의 AF 합초 포인트를 촬영 후 확인하고 싶을 때
- 표준 이미지 뷰어가 RAW 또는 CBZ(만화책 ZIP) 형식을 지원하지 않을 때

단일 실행 파일(`.exe`) 로 별도 설치 없이 바로 구동됩니다.

## 주요 기능

| 기능                 | 설명                                                          |
| -------------------- | ------------------------------------------------------------- |
| 압축파일 직접 열기   | ZIP / CBZ 내부 이미지를 압축 해제 없이 탐색                   |
| GPU 렌더링           | Rust + eframe/egui 기반, 부드러운 화면 전환                   |
| Canon CR3 RAW 지원   | rawler 라이브러리 활용, EXIF Orientation 자동 회전            |
| AF 포인트 오버레이   | Canon CR3의 합초(in-focus) AF 포인트 형광색 오버레이 (`A` 키) |
| 자동 Orientation fit | 가로 이미지 → 너비 기준, 세로 이미지 → 높이 기준 자동 맞춤    |
| 썸네일 스트립        | 하단 썸네일 바 (`T` 키), 앞뒤 25장 사전 로드                  |
| 아카이브 트리        | 좌측 파일 트리 패널 (`B` 키)                                  |
| 프리로드             | 메인 8장 + 썸네일 4장 병렬 사전 로드로 탐색 지연 최소화       |
| 최근 파일            | 우클릭 메뉴 → 최근 열었던 아카이브 목록                       |
| 정렬 전환            | 이름순 / 역순 / 랜덤 순환 (`S` 키)                            |
| 드래그 앤 드롭       | ZIP/CBZ 파일을 창에 드롭하여 즉시 열기                        |
| 전체화면             | `F11` 토글                                                    |

## 빠른 시작

### 빌드된 바이너리 사용

```
fast-image-viewer.exe
```

1. 실행 파일을 더블클릭하거나 터미널에서 위 명령 실행
2. 환영 화면에서 `O` 키를 눌러 ZIP/CBZ 파일 선택, 또는 파일을 창에 드래그 앤 드롭
3. `←` / `→` 방향키로 이미지 탐색

## 사용법

### 키보드 단축키

| 단축키    | 동작                                     |
| --------- | ---------------------------------------- |
| `O`       | 파일 열기 대화상자                       |
| `←` / `→` | 이전 / 다음 이미지                       |
| `I`       | EXIF 정보 오버레이 토글                  |
| `T`       | 썸네일 스트립 토글                       |
| `B`       | 아카이브 트리 패널 토글                  |
| `A`       | AF 포인트 오버레이 토글 (Canon CR3 전용) |
| `S`       | 정렬 순환 (이름순 → 역순 → 랜덤)         |
| `F11`     | 전체화면 전환                            |
| `Esc`     | 전체화면 해제                            |
| `Home`    | 뷰 초기화 (줌·패닝 리셋)                 |
| 마우스 휠 | 줌 인/아웃                               |

### 우클릭 메뉴

- 파일 열기
- 최근 파일 (서브메뉴)
- 정렬 방식 변경

### AF 포인트 오버레이 (Canon CR3)

Canon CR3 파일을 열고 `A` 키를 누르면 합초된 AF 포인트가 형광 녹색 박스와 십자선으로 표시됩니다.

- 표시 정보: AF 포인트 위치 + 중앙 십자마크
- 비활성 시 오버레이 완전 제거
- 힌트 텍스트에 `[A]AF ON` / `[A]AF` 상태 표시

## 성능

> 측정 환경: Canon EOS R7 CR3 137장, SSD, `bench_zip` 벤치마크 기준

| 항목                                            | v04           | v05              |
| ----------------------------------------------- | ------------- | ---------------- |
| 이미지 1장 평균 (zip 해제 + 디코딩 + RGBA 변환) | ~269ms        | ~264ms           |
| AF 추출 오버헤드                                | —             | +8ms (전체의 3%) |
| 최소 / 최대                                     | 208ms / 403ms | 217ms / 333ms    |
| GPU 텍스처 업로드                               | 15~20ms       | 15~20ms          |

**캐시 전략**:
- 메인 이미지: 앞뒤 8장 병렬 프리로드
- 썸네일: 현재 위치 기준 앞뒤 25장 사전 로드
- raw_cache: 장변 3840px 다운스케일 저장 (원본 ~129MB → ~39MB)

## FAQ / 문제 해결

**Q. 실행 시 "DLL을 찾을 수 없습니다" 오류가 납니다.**  
A. 배포용 빌드는 정적 링크로 컴파일되어 별도 DLL이 필요 없습니다. Visual C++ 런타임 오류라면 [Microsoft Visual C++ 재배포 가능 패키지 (x64)](https://aka.ms/vs/17/release/vc_redist.x64.exe)를 설치하세요.

**Q. CR3 파일이 열리지 않거나 어둡게 나옵니다.**  
A. rawler가 지원하지 않는 카메라 모델일 수 있습니다. 현재 테스트된 카메라는 Canon EOS R7입니다. 다른 모델은 동작을 보장하지 않습니다.

**Q. AF 포인트가 표시되지 않습니다 (`A` 키를 눌렀는데).**  
A. AF 오버레이는 Canon CR3 파일에서만 작동합니다. JPEG/PNG 등 다른 포맷은 지원하지 않습니다.

**Q. HDD에서 CR3 파일 로딩이 너무 느립니다.**  
A. CR3 RAW 디코딩은 SSD 환경에 최적화되어 있습니다(약 264ms/장). HDD 환경에서는 탐색 시 지연이 길어질 수 있습니다.

**Q. 특정 ZIP 파일이 열리지 않습니다.**  
A. v06부터 암호화된 ZIP/CBZ(ZipCrypto, AES-128, AES-256)를 지원합니다. 비밀번호가 틀리면 열리지 않으며, ZIP64 분할 아카이브 등 일부 비표준 케이스는 여전히 제한될 수 있습니다.

## 주의사항

- 이 프로그램은 **이미지 파일을 읽기 전용으로만 접근**합니다. 원본 파일을 수정하거나 삭제하지 않습니다.
- 대용량 ZIP 파일(수 GB 이상) 처리 시 초기 인덱싱에 시간이 걸릴 수 있습니다.
- GPU 렌더링 특성상, 원격 데스크톱(RDP) 환경에서는 렌더링 품질이나 속도가 저하될 수 있습니다.
- Canon CR3 AF 포인트 파싱은 Canon EOS R7 기준으로 구현되었습니다. 타 기종에서의 정확성은 보장되지 않습니다.
- 소프트웨어 오동작으로 인한 손해에 대해 개발자는 책임을 지지 않습니다.
