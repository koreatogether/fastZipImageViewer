# fastImageViewer

압축파일(ZIP/CBZ) 속 이미지를 빠르게 탐색하는 GPU 가속 이미지 뷰어.  
Canon CR3 RAW 파일 지원, AF 포인트 오버레이, 애니메이션 GIF 재생, 손상된 압축파일 내결함성 복구를 포함합니다.

## 소개

fastImageViewer는 다음 문제를 해결하기 위해 만든 도구입니다.

- 수백 장의 RAW/JPEG가 들어 있는 ZIP 아카이브를 외부 압축 해제 없이 바로 열어 빠르게 넘겨봐야 할 때
- Canon CR3(RAW) 파일의 AF 합초 포인트를 촬영 후 확인하고 싶을 때
- 표준 이미지 뷰어가 RAW 또는 CBZ(만화책 ZIP) 형식을 지원하지 않을 때

단일 실행 파일(`.exe`) 로 별도 설치 없이 바로 구동됩니다.

현재 공개 배포 산출물은 저장소 루트의 `fast-image-viewer_v09.exe`이며, v09에서는 손상된 ZIP/CBZ 복구성 개선과 UI 응답성 보강이 반영되었습니다. 이전 배포본은 `oldEXE/`에 아카이브되어 있습니다.

| 백그라운드 로딩      | 아카이브 열기를 분리해 UI 응답성을 유지                          |

| 애니메이션 GIF       | ZIP/CBZ 내부 GIF를 프레임 단위로 자동 재생                       |

| 프리로드             | 메인 8장 + 썸네일 4장 병렬 사전 로드와 캐시 최적화로 지연 최소화 |
| 최근 파일            | 우클릭 메뉴 → 최근 열었던 아카이브 목록                          |
| 정렬 전환            | 이름순 / 역순 / 랜덤 순환 (`S` 키)                               |
| 드래그 앤 드롭       | ZIP/CBZ 파일을 창에 드롭하여 즉시 열기                           |
| 전체화면             | `F11` 토글                                                       |

## 빠른 시작

### 빌드된 바이너리 사용

```
fast-image-viewer_v09.exe
```

1. 저장소 루트에서 `fast-image-viewer_v09.exe`를 실행합니다.
2. 환영 화면에서 `O` 키를 눌러 ZIP/CBZ 파일을 선택하거나, 파일을 창에 드래그 앤 드롭합니다.
3. `←` / `→` 방향키로 이미지 탐색을 진행합니다.
4. 이전 배포본이 필요하면 `oldEXE/README.MD`를 참고합니다.

### 우클릭 메뉴

- 파일 열기
- 최근 파일 (서브메뉴)
- 정렬 방식 변경

### AF 포인트 오버레이 (Canon CR3)

Canon CR3 파일을 열고 `A` 키를 누르면 합초된 AF 포인트가 형광 녹색 박스와 십자선으로 표시됩니다.

- 표시 정보: AF 포인트 위치 + 중앙 십자마크
- 비활성 시 오버레이 완전 제거
- 힌트 텍스트에 `[A]AF ON` / `[A]AF` 상태 표시

## 성능

> 참고: v04/v05 수치는 `progressLog.md`에 남아 있는 `_W8A1244.zip`(CR3 137장, SSD) 기준 기록값이고, v08/v09 수치는 `src/v01/test_images/test_images.zip`(72파일)으로 다시 측정한 값입니다. 데이터셋이 다르므로 절대 비교보다는 버전 추세 참고용으로 봐주세요.

v08의 mmap 기반 ZIP 접근 캐시와 GIF 처리 최적화는 v09에도 유지됩니다. v09에서는 여기에 손상 압축파일 salvage 경로와 백그라운드 아카이브 로딩을 더해, 깨진 파일에서도 가능한 범위의 이미지를 더 안정적으로 보여주도록 정리했습니다.

| 버전 | 측정 대상                  | 평균 총 처리시간 | AF 추출 평균 | 비고                           |
| ---- | -------------------------- | ---------------- | ------------ | ------------------------------ |
| v04  | `_W8A1244.zip` / CR3 137장 | ~269ms           | -            | zip 해제 + 디코딩 + RGBA 변환  |
| v05  | `_W8A1244.zip` / CR3 137장 | ~264ms           | ~8ms         | AF 추출 추가                   |
| v08  | `test_images.zip` / 72파일 | ~250ms           | ~9ms         | mmap 캐시 + GIF 최적화         |
| v09  | `test_images.zip` / 72파일 | ~223ms           | ~8ms         | salvage 폴백 + 백그라운드 로딩 |

v09는 같은 `test_images.zip` 기준에서 v08보다 평균 총 처리시간이 약 27ms 짧았습니다.

**캐시 전략**:
- 메인 이미지: 앞뒤 8장 병렬 프리로드
- 썸네일: 현재 위치 기준 앞뒤 25장 사전 로드
- raw_cache: 장변 3840px 다운스케일 저장 (원본 ~129MB → ~39MB)

## FAQ / 문제 해결

**Q. 실행 시 "DLL을 찾을 수 없습니다" 오류가 납니다.**  
A. 배포용 빌드는 정적 링크로 컴파일되어 별도 DLL이 필요 없습니다. Visual C++ 런타임 오류라면 [Microsoft Visual C++ 재배포 가능 패키지 (x64)](https://aka.ms/vs/17/release/vc_redist.x64.exe)를 설치하세요.

**Q. CR3 파일이 열리지 않거나 어둡게 나옵니다.**  
A. rawler가 지원하지 않는 카메라 모델일 수 있습니다. 현재 테스트된 카메라는 Canon EOS R7입니다. 다른 모델은 동작을 보장하지 않습니다.

**Q. AF 포인트가 표시되지 않습니다 (`A` 키를 눌렀는데).**  
A. AF 오버레이는 Canon CR3 파일에서만 작동합니다. JPEG/PNG 등 다른 포맷은 지원하지 않습니다.

**Q. HDD에서 CR3 파일 로딩이 너무 느립니다.**  
A. CR3 RAW 디코딩은 SSD 환경에 최적화되어 있습니다(약 264ms/장). HDD 환경에서는 탐색 시 지연이 길어질 수 있습니다.

**Q. 특정 ZIP 파일이 열리지 않습니다.**  
A. 암호화된 ZIP/CBZ 파일을 지원하며, v09부터는 손상된 아카이브도 가능한 범위에서 이미지를 복원합니다. 다만 비밀번호가 틀리거나 구조가 지나치게 손상된 파일은 여전히 열리지 않을 수 있습니다.

## 주의사항

- 이 프로그램은 **이미지 파일을 읽기 전용으로만 접근**합니다. 원본 파일을 수정하거나 삭제하지 않습니다.
- 대용량 ZIP 파일(수 GB 이상) 처리 시 초기 인덱싱에 시간이 걸릴 수 있습니다.
- GPU 렌더링 특성상, 원격 데스크톱(RDP) 환경에서는 렌더링 품질이나 속도가 저하될 수 있습니다.
